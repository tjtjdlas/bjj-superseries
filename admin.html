[ 01 ] 분류 항목 설정

이름·소속 외에 관리할 분류 항목입니다. 기본값은 체급/연령/성별이며, 자유롭게 추가·삭제·이름 변경이 가능합니다. 항목 순서가 아래 엑셀 붙여넣기 열 순서가 됩니다.

[ 02 ] 엑셀 붙여넣기로 등록

엑셀에서 아래 순서의 열을 선택해 복사(Ctrl+C)한 뒤, 텍스트 상자에 붙여넣으세요(Ctrl+V). 헤더(제목) 행은 제외하고 데이터만 복사해 주세요.

[ 03 ] 현재 등록된 선수 명단

셀을 직접 클릭해서 수정할 수 있습니다. 변경사항은 이 브라우저에 자동 저장됩니다 (아래 내보내기 전까지는 실제 사이트에는 반영되지 않습니다).

[ 04 ] GitHub 자동 반영 설정

GitHub 저장소에 쓰기 권한이 있는 Fine-grained Personal Access Token을 발급받아 아래에 입력하면, [05]의 "지금 사이트에 반영하기" 버튼으로 파일을 직접 업로드하지 않고 바로 실제 사이트에 반영할 수 있습니다. 토큰은 이 브라우저에만 저장되며(다른 기기·브라우저에는 없음), 절대 다른 사람과 공유하거나 공용 컴퓨터에 저장하지 마세요.

발급 방법: GitHub 우측 상단 프로필 → Settings → 좌측 맨 아래 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
· Repository access: Only select repositories → bjj-superseries 선택
· Permissions: Contents → Read and write 로 설정 (다른 권한은 전부 No access로 둡니다)
· 생성된 토큰(github_pat_...)을 복사해 아래에 붙여넣으세요.

[ 05 ] 게시 / 불러오기

[04]에서 토큰을 저장했다면 지금 사이트에 반영하기 버튼 한 번으로 실제 사이트에 바로 올라갑니다. 토큰을 아직 안 쓰신다면 JSON 파일을 다운로드해 저장소의 assets/data/roster.json에 직접 덮어써도 됩니다.

작업 내용은 이 브라우저의 로컬 저장소에만 보관됩니다. 반영하기 전까지는 실제 사이트에 영향이 없습니다.
