SPYDER / 선수 명단 관리자 ← 사이트로 돌아가기

[ 01 ] 분류 항목 설정

이름·소속 외에 관리할 분류 항목입니다. 기본값은 체급/연령/성별이며, 자유롭게 추가·삭제·이름 변경이 가능합니다. 항목 순서가 아래 엑셀 붙여넣기 열 순서가 됩니다.

[ 02 ] 엑셀 붙여넣기로 등록

엑셀에서 아래 순서의 열을 선택해 복사(Ctrl+C)한 뒤, 텍스트 상자에 붙여넣으세요(Ctrl+V). 헤더(제목) 행은 제외하고 데이터만 복사해 주세요.

[ 03 ] 현재 등록된 선수 명단

셀을 직접 클릭해서 수정할 수 있습니다. 변경사항은 이 브라우저에 자동 저장됩니다 (아래 내보내기 전까지는 실제 사이트에는 반영되지 않습니다).

[ 04 ] 게시 / 불러오기

아래 JSON 파일 다운로드로 받은 roster.json 파일을 웹사이트 저장소의 assets/data/roster.json에 덮어써야 실제 방문자에게 반영됩니다.

작업 내용은 이 브라우저의 로컬 저장소에만 보관됩니다. 다른 기기나 브라우저에서는 보이지 않으니, 작업이 끝나면 반드시 JSON을 다운로드해 저장소에 반영해 주세요.
